#  Experimental Data Management System (EDMS)

##  Overview
The Experimental Data Management System (EDMS) is a research-oriented software system developed to manage, process, analyze, and intelligently evaluate experimental datasets.

The project evolved progressively through six milestones, integrating:

- Object-Oriented Programming (OOP)
- Design Patterns
- Functional Programming
- Concurrency & Parallel Computing
- Statistical Analysis
- Intelligent Anomaly Detection

The system uses a wine quality dataset as a case study for experimental data analysis.

---

#  Project Objectives

The main objectives of this project are:

- To model experimental data using object-oriented principles
- To implement scalable and maintainable software architecture
- To improve performance using concurrency techniques
- To automate statistical analysis of datasets
- To integrate intelligent anomaly detection using Z-score statistics
